# Разработка интернет-приложений (РИП) 2025

## Прогноз роста объема данных

Разработанное приложение позволит аналитикам создавать и формировать заявки на прогноз роста объёма данных с учётом выбранных факторов и заданного периода расчёта, а менеджерам данных — утверждать либо отклонять данные заявки.
